# CryptoPulse 🚀

CryptoPulse is a clean and modern cryptocurrency dashboard that displays real-time crypto asset data such as prices, market changes, and a watchlist.  
The UI is inspired by modern fintech dashboards and designed based on a Figma layout.

---

## 🔗 Live Demo
https://mysterious20.github.io/cryptopulse/

---

## ✨ Features
- Real-time cryptocurrency data using public API
- Watchlist with star (wishlist) functionality
- Asset cards with logo, name, symbol, and price change
- Responsive and modern UI
- Horizontally scrollable watchlist
- Clean deployment using GitHub Pages

---

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- Public Crypto API
- GitHub Pages (Deployment)

## 🚀 Deployment
The project is deployed using **GitHub Pages**.

Steps:
1. Push code to GitHub repository
2. Enable GitHub Pages from `main` branch
3. Site auto-deploys on every push
